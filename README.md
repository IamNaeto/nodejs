# Node.js Learning Topics

Welcome to my Node.js Learning Topics Repository! This repository covers a wide range of Node.js topics from introductory to advanced levels, making it a comprehensive resource for learning and mastering Node.js development.

## Getting Started

To get started with this repository, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/IamNaeto/nodejs.git
   ```

2. **Navigate to the topic of interest:**
   Explore the repository's directory structure to find topics categorized by complexity and functionality.

3. **Read the README and code comments:**
   Each topic folder contains a .txt file with instructions and explanations and script files with codes and comments for further explanations. Dive into the code and comments to understand the concepts better.

## Repository Structure

The repository is structured into various folders, each covering a specific Node.js topic. Here are some of the key folders and their contents:

- `01_hello-world/`: Introduction to Node.js, executing my first hello world javascript program outside the browser.
- `02-07_modules/`: Modules(user defined) in Node.js, asynchronous programming with callbacks, Promises, and async/await.
- `08-26_built-in-modules/`: Modules(user defined) in Node.jsAsynchronous programming with callbacks, Promises, and async/await.
- `27_web-framework/`: Introduction to building web applications using the Express.js framework.
- `28-38_node.js-internals/`: Libuv, thread_pool, thread_pool size, network_io, event loop, event loop queues.
- `39-47_npm/`: Essential for building medium to large scale app with Nodejs.
- `48-50-cli-tools/`: Using node.js to build a command line interface tool.
- `51-54_misc/`: Cluster module, worker threads module, deploying nodejs app, wrapping up

## Features

- Comprehensive coverage of Node.js concepts and best practices.
- Hands-on examples and code snippets for practical understanding.
- Incremental learning path from basic to advanced topics.
- Clear explanations and text files (.txt) for each topic.
